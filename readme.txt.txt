To run the project .
1. clone the reporisitry on your local machine 
2. open the backend folder in your VS code and run the command 'npm i'
3. after that run the command 'nodemon '.\server.js';to run the server
4. Now your server is running
5. Switch to frontend folder and go triptastic directory .
6. run the command 'npm i'
7. run 'npm start'
8. project is running now
